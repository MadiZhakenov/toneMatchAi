# ToneMatch AI - Design Prototype

Этот прототип позволяет визуализировать и тестировать дизайн UI плагина ToneMatch AI перед имплементацией в VST3.

## Как использовать

1. Откройте файл `design_prototype.html` в браузере
2. Изучите дизайн и взаимодействуйте с элементами:
   - Крутите регуляторы (knobs) мышью
   - Кликайте на слайдеры для изменения значений
   - Наводите курсор на кнопки для просмотра эффектов

## Структура файлов

- `design_prototype.html` - HTML разметка UI
- `design_prototype.css` - Стили и визуальное оформление
- `design_prototype.js` - Интерактивность (анимации, взаимодействие)

## Элементы дизайна

### Цветовая схема
- Фон: градиент от #2b2b2b до #1a1a1a
- Акцент: #ffb249 (оранжевый)
- Текст: #e0e0e0 (светло-серый)
- Поверхности: #2a2a2a (темно-серый)

### Шрифт
- Используется Manrope (Google Fonts) - SemiBold (600)

### Компоненты
1. **Верхняя панель** - заголовок, транспорт, пресет, выходной уровень
2. **Основной раздел** - INPUT, центральная область с логотипом, кнопками, регуляторами, OUTPUT
3. **Секция эффектов** - панели Delay и Reverb с чекбоксами, слайдерами и регуляторами

## Изменение дизайна

Для изменения дизайна редактируйте соответствующие файлы:
- **Цвета**: измените значения в `design_prototype.css` (ищите комментарии с цветами)
- **Размеры**: измените значения width/height/padding в CSS
- **Расположение**: измените flex/grid свойства в CSS
- **Текст**: измените содержимое в `design_prototype.html`

## После утверждения дизайна

Когда дизайн будет утвержден, изменения будут перенесены в:
- `plugin/Source/UI/StyleSheet.h` - стили для VST плагина
- `plugin/Source/PluginEditor.cpp` - layout и компоненты

## Примечания

- Прототип использует Google Fonts для Manrope (в продакшене используется встроенный шрифт)
- Некоторые эффекты (тени, градиенты) могут отличаться в финальной версии VST
- Размеры могут быть скорректированы для оптимальной работы в DAW
